# Currency-Converter-
Aplicación Web destinada a obtener la tasa de USD, VED, COP y BTC actual para convertir sin necesidad de acceder a cuentas de terceros. 

## Pasos

Para poder tener acceso a la app, debes seguir los siguiente pasos:

1. Hacer fork del proyecto: https://github.com/hecatonquiro-yolo/Currency-Converter-
1. Instalar los dependencias con el comando "pip install requirements.txt"
2. Instalar las dependencias que están en los respectivos package.json de /frontend y /backend
3. Ejecutar con NodeJs el archivo <scraper.js> con el comando "node scraper.js" en la carpeta Raíz.
4. Activar el servidor con NodeJs el archivo <app.js> en la carpeta Para activar el servidor y conectarse a las bases de datos.
5. Ejecutar el comando: "npm run start" desde la carpeta /frontend
